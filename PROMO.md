## 4. VSCode Marketplace SEO 改善案

### 現在の keywords（package.json）
```json
"keywords": [
  "tyranoscript",
  "tyranobuilder",
  "visual novel",
  "ティラノスクリプト",
  "ティラノビルダー"
]
```

### 改善案
Marketplace は最大 5 キーワードまで。現在の5つは良い選択だが、
競合 tyranosyntax のキーワード（ADV, KAG, tyrano, ノベル, ゲーム）との
差別化を考慮すると、以下も検討：

**代替案 A — 「ノベルゲーム」を入れる**
```json
"keywords": [
  "tyranoscript",
  "tyranobuilder",
  "ノベルゲーム",
  "ティラノスクリプト",
  "ティラノビルダー"
]
```
→ 日本語ユーザーが「ノベルゲーム」で検索するケースをカバー

**代替案 B — 英語キーワードを増やす**
```json
"keywords": [
  "tyranoscript",
  "tyranobuilder",
  "visual novel",
  "game engine",
  "ティラノスクリプト"
]
```
→ 英語圏ユーザーへのリーチを強化

### README の SEO 改善
README.md は Marketplace の詳細ページに表示される。以下を追加検討：
- 機能ごとのスクリーンショット / GIF（視覚的アピール、Marketplace 評価に直結）
- バッジ（バージョン、ライセンス、ダウンロード数）
- 「TyranoScript」「TyranoBuilder」「ノベルゲーム」「visual novel」を
  自然な文脈で README 内に含める
- 英語 README をデフォルトにし、日本語は折りたたみまたは別ファイルで提供
  （Marketplace のデフォルト表示が英語のため、グローバルリーチが上がる）

### displayName の SEO
現在: `TyranoCode — TyranoScript Development Suite`
→ 良い。「TyranoScript」が含まれており検索にヒットしやすい。

### description の SEO
現在: `Professional development tools for TyranoScript / TyranoBuilder: debugger, analyzer, flow graph, auto-test, profiler, and more.`
→ 良い。主要キーワードが自然に含まれている。
