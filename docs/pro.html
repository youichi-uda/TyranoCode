Pro 機能の詳細

🐞

デバッガー PRO

ブレークポイントの設定、ステップ実行、変数のリアルタイム監視。 ゲームを実行しながら TyranoScript の動作を直接確認できます。

📊

フローグラフ PRO

シナリオ間の jump / call / glink / button の遷移を自動解析し、 インタラクティブなフローチャートで表示。全体構造を一目で把握できます。

🧪

全ルートテスト PRO

ゲーム内の全分岐ルートを自動探索。到達不能な選択肢や 無限ループがないかを一括チェックできます。

⏱

パフォーマンスプロファイラー PRO

各シーンのタグ数・リソース読み込み数・トランジション回数を分析。 ボトルネックとなるシーンを特定し、最適化の指針を提供します。
